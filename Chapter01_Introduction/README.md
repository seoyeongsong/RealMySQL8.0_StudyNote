### 1.1 MySQL 소개

- MySQL은 소스가 공개된 오픈소스 데이터 베이스이다.
- 현재 Oracle에서 인수되었으나 라이센스에 큰 변화는 없음
- MySQL 8.0 버전부터는 상용 DBMS가 가진 기능들이 장착되기 시작
- 엔터프라이즈 에디션은 소스코드는 공개되지 않음
- 커뮤니티 에디션은 여전히 소스코드가 공개된 상태
