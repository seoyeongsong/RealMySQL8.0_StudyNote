### 6.1 페이지 압축
- Transparent Page Compression
```sequence
SQL Server->디스크: 데이터 페이지 압축 저장
디스크-->SQL Server: 압축 해제하여 읽기
```
- InnoDB의 버퍼 풀에 데이터 페이지가 한 번 저장되면 스토리지 엔진은 압축이 해제된 상태로만 데이터 페이지를 관리
